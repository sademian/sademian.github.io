Programming Projects

Below are a few of the programming projects I have worked on so far; you can click on some of the titles to learn more about them.

Synaptograph (Capstone Project) - Python

A desktop application that creates 3D models from EEG brain scans. The 3D models are saved as .stl files and can be printed using a 3D printer.

iOS GPS Game - Swift

Location based game where players are given clues for locations inside a theme park and earn points for visiting these locations.

Mondrian Art Generator - JavaScript

Randomly generates Mondrian style art using a divide and conquer approach.

Simple PL/0 Compiler - C

Reads a program written in PL/0 and compiles and runs it on a virtual machine. This program included a parser, lexical analyzer, intermediate code generator, and a virtual machine.

Attacking Knights - Java

Takes a list of coordinate strings for knights on an arbitrarily large square chess board, and determines whether any of them can attack one another.

Topological Sort - Java

Takes a directed graph and uses backtracking to generate all valid topological sorts for that graph.

Efficient Fibonacci Calculator - C

O(n) Fibonacci calculator, it also uses arrays of integers to represent numbers greater than 64-bit integers.

Hash table - C

A Hash table implementation with linear and quadratic probing. It supports insertion, search, deletion, and expands dynamically when it gets half full.

Chess Moves Interpreter - C

A program that parses strings written in algebraic chess notation and prints a graphical representation of the chess board after each move.
