Smaller Class Assignments


Device Driver - C

A simple kernel module character-mode Linux device driver that stores bytes written to it up to a constant buffer size, allows them to be read back out in FIFO fashion, and removes them from the buffer as they are read back out


Process Scheduler [Team of 2] - C

A simple operating system process scheduler that takes a list of processes and schedules them using First-Come First-Served, preemptive Shortest Job First, and Round-Robin algorithms as for single processors


Attacking Knights - Java

Takes a list of coordinate strings for knights on an arbitrarily large square chess board, and determines whether any of them can attack one another


Sneaky Queens - Java

Takes a list of coordinate strings for queens on an arbitrarily large square chess board, and determines whether any of them can attack one another with linear runtime


Constrained Topological Sort - Java

Given an arbitrary directed graph and vertices x and y, determines if the graph has a valid topological sort in which x comes before y


All Topological Sorts - Java

Takes a directed graph and uses backtracking to generate all valid topological sorts for that graph


Efficient Fibonacci Calculator - C

O(n) Fibonacci calculator, it also uses arrays of integers to represent numbers greater than 64-bit integers


Hash table - C

A Hash table implementation with linear and quadratic probing. It supports insertion, search, deletion, and expands dynamically when it gets half full


Chess Moves Interpreter - C

A program that parses strings written in algebraic chess notation and prints a graphical representation of the chess board after each move
