Wizard Hunt

A scavengar hunt iOS game to be played at the Wizarding World of Harry Potter at Universal Studios.
This app is not and will not be published. It was just a fun personal project that I wanted to work on!

Home Page

Start a new game or continue with a saved game

Clue Selection Page

Select one of the available clues. The clue icon chenges to a star once the clue is unlocked

Clue Page

Read the clue and try to find the location

Map View Page

You can see the locations and names of solved clues

Solved Clues Page

You can also see a list of the solved clues
