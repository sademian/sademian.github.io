Week 6 - Generating Windows and Doors

This week I worked on adding windows and doors.

Adding Window and Door Mesh classes

I derived two classes from my Modular Piece class for the doors and winows. The reason I decided to derive classes instead of making a type enum is because I wanted to use unreal's class parameter filtering

Procedural Windows/Door Generation

Next, I worked on generating the windows and doors layout. Currently, the algorithm I wrote attempts to place a window at a 1/3 chance most of the time. If two windows are placed next to each other then a door is chosen next to make sure there are no possibility of generating 3 windows next to each other, on the other hand if we have two consecutive walls then the chance of having a window next increases to 1/2.
